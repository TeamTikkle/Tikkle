## 주요 페이지
- Tikkle
  
  내가 갖고 있는 티끌판 리스트를 보여줍니다. 

  생성하기를 누르면, 새로운 티끌판을 생성할 수 있습니다. 

- Feed

   다른 사람들의 티끌판 리스트를 둘러볼 수 있습니다. 
 
- MyPage

  나의 기본 정보를 조회할 수 있습니다.
